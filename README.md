# FreeDNS
Send a dynamic IP to another dynamic IP

Once you have installed DD-WRT to your router you need to 
1 ) Upload the SH file with scp from you linux PC to you router
2 ) Create a user in your server 
3 ) Go to bit.do and create a link to your server destination 
4 ) Modify the page destination in the script
5 ) create a file called instalar.xxx in the same folder as the script
6 ) run the script
7 ) Answer the questions
